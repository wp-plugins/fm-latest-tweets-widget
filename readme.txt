== Description ==

Display your latest tweets form your twitter account by using this plugin on your website.

More plugin by this author: [Responsive facebook like box Translate](http://wordpress.org/plugins/responsive-facebook-like-box/)


Go here for screenshots : http://freewebmentor.com/2015/03/fm-latest-tweets-widget.html


== Installation ==

1. Unzip folder to the `/wp-content/plugins/` directory
2. Log into your Wordpress admin and activate the 'Latest Tweets widget' plugin through the 'Plugins' menu
3. Once the plugin is installed and enabled you can find it in your widget and then follows the steps.

Widgets can be created by logging into Twitter and:

Go to: https://twitter.com/settings/widgets
Press the 'Create new' button
Fill in the settings
Hit the 'Create widget' button
You widget ID will be visible in the code text box:
